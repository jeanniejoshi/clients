Notes for Bret / Development

Nav inheritance: The My Why page will automatically inherit the existing HPVCA header and footer through WordPress/Divi. Our code does not include a nav — it starts where the nav ends.

Palette mismatch: The current site uses a darker maroon-wine purple (~#4a0e2a range) for hero banners. The official HPVCA brand guide specifies #914193 (Pantone 259C). Our code uses the official palette. If the team wants to match the existing site purple instead, it's a single CSS variable swap: --purple: #4a0e2a;. Recommendation: keep the official palette and let the My Why page set the standard for the site refresh.

Integration: Two paths — (1) use these HTML files as visual reference and rebuild natively in Divi 5, or (2) paste the code directly into Divi Code Modules for faster launch. CSS variables, responsive breakpoints, and lightbox JS all work standalone.
